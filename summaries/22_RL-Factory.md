# RL-Factory: A Plug-and-Play Reinforcement Learning Post-Training Framework
#Hardware_Topics #Training #GPU-side #Scenarios #GUI-agent

## Summary
RL-Factory is a specialized reinforcement learning post-training framework designed for Agentic Learning, focusing on environment decoupling and tool integration. It achieves 6.8x increased training throughput while maintaining exceptional efficiency and stability in multi-turn tool usage scenarios.

## Key Technical Innovations

### 1. Environment Decoupling Architecture [Environments_Computation][System_/_Runtime]

![RL-Factory Architecture](https://github.com/Simple-Efficient/RL-Factory/raw/main/docs/architecture.png)

**Figure 1**: RL-Factory's modular architecture with decoupled environment management

- **Environment Separation**: Revolutionary approach separating environment logic from RL post-training processes
- **Tool Integration Layer**: Native support for external tool setup and reward configuration
- **Plug-and-Play Design**: Modular framework enabling easy integration with existing systems
- **Multi-Modal Support**: Comprehensive support for vision-language models in agentic tasks
